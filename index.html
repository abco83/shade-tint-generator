About

Over

This “tool” calculates the shades and tints for any color, in addition it tries to make brighter versions of the color as well. When the calculated color is “corrected” e.g. the rgb value was set to max or min because it exceeded that value, the box will have an ugly border.

Deze “tool” rekent de verschillende tinten voor elke kleur uit, behalve dat probeert het ook een helderdere versie van de kleur te maken. Als de uitgerekende kleur “gecorrigeerd” werd als in de waarde was boven of onder het maximum, heeft het vak een de lelijke rand.

Functionality:

Funties:

  • Copy any Hex or rgb code into the box above "Get HEX".
  • Input from rgb numbers, color picker or type in hex.
  • Click on any colored button to copy it’s hex code to clipboard.
  • Click on “Get RGB” to copy the rgb code to clipboard.
  • Information about the meaning of colors. ↓
  • Change the text or border color.
  • Enable/disable recalculating from clicks on shades and tints.
  • Saves the calculated decimals while clicking through.
  • Calculate the negative of the current color.
  • 2 x 20 examples to click on.
  • Random color selector.
  • Kopieer een Hex of rgb code in het vak boven "Get HEX".
  • Invoer van rgb nummers, kleurkiezer of tik in een hex.
  • Klik op een kleur om de hex code op het klembord te plaatsen.
  • Klik op “Get RGB” om de rbg code op het klembord te plaatsen.
  • Informatie over de betekenis van kleuren. ↓
  • Verander de kleur van de text of de rand.
  • Zet doorklikken van de shades en tints aan of uit.
  • Bewaard de uitgerekende decimalen bij het doorklikken.
  • Reken de negatieve versie van de kleuren uit.
  • 2 x 20 voorbeelden om op te klikken.
  • Willekeurige kleur kiezer.
